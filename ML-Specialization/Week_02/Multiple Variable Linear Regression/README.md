# Multiple Variable Linear Regression From Scratch

This project demonstrates the implementation of Multiple Variable Linear Regression using Python and NumPy.

Concepts Covered

- Linear regression with multiple features
- Vectorization using NumPy
- Cost function (Mean Squared Error)
- Gradient Descent optimization

Dataset

A small synthetic dataset is used to demonstrate the algorithm.

Technologies

Python
NumPy
Jupyter Notebook

Purpose

This project is built for learning and understanding the fundamentals of machine learning algorithms without using ML libraries.
