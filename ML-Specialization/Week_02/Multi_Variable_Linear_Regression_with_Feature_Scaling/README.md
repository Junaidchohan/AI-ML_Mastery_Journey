# Multi-variable Linear Regression with Feature Scaling

## Project Overview
This project demonstrates how to build a **linear regression model with multiple features** to predict housing prices.
It is beginner-friendly, fully explained, and easy to revisit for learning purposes.

### Features
- **Input features:** Size (sqft), Number of bedrooms, Number of floors, Age of home
- **Target variable:** Price (in $1000s)
- **Implemented from scratch:** Gradient descent for multiple variables
- **Feature scaling:** Z-score normalization for faster convergence
- **Visualization:** Feature vs target plots, convergence plots, predictions vs actual
- **Prediction:** Estimate price for new houses

- Python 3
- NumPy
- Matplotlib
